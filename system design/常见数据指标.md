常见数据指标
- L1 缓存 ：1ns
- L2 缓存：3ns
- Mutex Lock：17ns
- Main memory reference：100ns
- zippy 压缩1kb：2000ns
- 从内存中顺序读取1M数据：10000ns，0.010ms
- 10G带宽，发送2kb数据：1600ns
- SSD 4kb随机读取：20000ns，0.020ms
- 顺序读SSD 1MB：1ms，相当于1G/s
- 磁盘1M顺序读：5ms，相当于200M/s
- 1M数据，1G带宽：10ms
- Disk seek：10ms
- TCP 数据包往返周期：150ms（大陆间）

磁盘顺序读取1M数据的耗时是SSD的5倍
